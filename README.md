# Customer Sales Data Cleaning & Visualization Project

## Project Overview

This project focuses on cleaning, analyzing, and visualizing customer sales data using Python.

The objective is to transform raw data into meaningful business insights through data preprocessing and visualization techniques.

---

## Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Project Workflow

The project follows these steps:

1. Dataset Creation
2. Data Understanding
3. Data Cleaning
4. Missing Value Treatment
5. Duplicate Removal
6. Outlier Detection
7. Exploratory Data Analysis
8. Data Visualization
9. Business Insights

---

## Data Cleaning Performed

The following preprocessing techniques were applied:

- Handled missing values using statistical methods
- Removed duplicate records
- Detected and treated outliers using IQR method
- Prepared clean dataset for analysis

---

## Exploratory Data Analysis

The following analyses were performed:

- Category-wise sales analysis
- City-wise sales performance
- Product performance analysis
- Customer demographic analysis
- Payment method analysis
- Rating analysis
- Correlation analysis

---

## Key Insights

- Identified top-performing product categories
- Analyzed customer purchasing patterns
- Compared sales performance across cities
- Studied relationships between numerical variables
